# Flight Price Intelligence: Data Analysis and Predictive Modelling for Airfare Optimisation

## 📌 Project Overview
Flight ticket prices are highly dynamic and influenced by multiple factors such as booking time, airline, and travel routes. This project combines data analysis and machine learning techniques to analyse historical flight data and predict future ticket prices, enabling more informed and cost-effective travel decisions.

---

## 🎯 Objective
The objective of this project is to develop a robust analytics and predictive modelling solution to:
- Identify key factors affecting flight prices  
- Analyse pricing trends and patterns  
- Forecast airfare using machine learning  
- Support data-driven decision-making for optimal booking  

---

## ✨ Key Features
- End-to-end data pipeline (data collection, cleaning, preprocessing)  
- Exploratory Data Analysis (EDA) to uncover trends and insights  
- Feature engineering to improve model performance  
- Implementation of regression-based machine learning models  
- Model evaluation using RMSE and R² metrics  

---

## 📊 Dataset
The dataset was collected via web scraping from the KAYAK platform and includes:
- Airline name  
- Source and destination  
- Departure and arrival time  
- Journey duration  
- Booking date and travel date  
- Ticket price  

This dataset enables both analytical exploration and predictive modelling of airfare trends.
